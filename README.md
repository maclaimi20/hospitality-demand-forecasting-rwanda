# Hospitality Consumable Demand Forecasting

## Overview
This project predicts demand for hospitality consumables in Rwandan TVET institutions using machine learning.

## Dataset
The dataset includes:
- Date
- Type of consumable
- Quantity used
- Number of students
- Events/season

## Model Used
- Linear Regression
- Random Forest

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the notebook
